Sistema de Gestão Bibliotecaria

O sistema de Gestão Bibliotecaria é um sistema web densevolvido na linguagem PHP, e base de dados MYSQL, o sistema facilita o controlo entrada e saída de livros e outro material disponibilizado pela bibioteca escolar, conrole de usuarios e emissao de relatorios tornando processos mais eficais e trazendo beneficios ao alunos e para escola

  • Linguagem: PHP
  • Base de dados: MySQL
  • Outros: JS, CSS, e Bootstrap
